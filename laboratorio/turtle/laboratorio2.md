### Laboratorio 2 (Turtle)

Se dibuja la tortuga segun las instrucciones con dos vectores x[ ] e y[ ], y se le dio movimiento con el teclado cuyas teclas son las siguientes:
* w => UP
* s => DOWN
* a => LEFT
* d => RIGHT
* q => LEFT ROTATION
* e => RIGHT ROTATION
* r => UP ROTATION
* f => DOWN ROTATION
* z => MAXIMIZATION
* x => MINIMIZATION
